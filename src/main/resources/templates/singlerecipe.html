Soup Recipe name #1

  • Вода - 2.5л
  • Картофель - 300гр
  • Морковь - 300гр
  • Болгарский перец - 200гр
  • Броколли - 500гр
  • Cоль/специи - по вкусу
1. Почистить, порезать картофель и морковь. Варить минут 40. После отваривания дать постоять минут 10-15 остыть.
2. Измельчить отваренные овощи блендером.
3. Добавить нарезанные болгарский перец. Варить минут 30.
4. Добавить броколли. Варить еще минут 20.
5. Добавить соль/специи по вкусу.
Приятного аппетита.
